How to Use

1. Include the CSS

<link rel="stylesheet" href="unified-style.css">

2. Use the Header Template

<?php include 'header-unified.php'; ?>

3. Use the Footer Template

<?php include '../footer-unified.php'; ?>

4. Apply Consistent Classes

  • Use .card for all card components
  • Use .btn with modifiers for buttons
  • Use .grid with .grid-2, .grid-3, .grid-4 for layouts
  • Use .space-y-6 for consistent vertical spacing
  • Use design tokens (CSS variables) for colors and spacing
